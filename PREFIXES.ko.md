# Relic Forge — 접두사 가이드

**언어:** [English](PREFIXES.en.md) · 한국어 · [简体中文](PREFIXES.zh.md)

강화된 유물에 붙는 모든 접두사입니다. 접두사는 **런 시드로 결정**되며(같은 시드 = 같은 결과), 시작 유물은 자동 제외됩니다. 기본적으로 유물의 약 **40%만** 접두사를 받습니다(ModConfig 슬라이더로 조절).

> **비중** 열은 **접두사가 붙을 때** 각 접두사가 뽑힐 비중입니다. 실제 유물당 확률 ≈ 40% × 비중.

---

## 1. 수치 접두사

유물의 수치 효과를 올리거나 내립니다. 툴팁에 증감이 색으로 표시됩니다(초록 증가 / 빨강 감소).

| 접두사 | 효과 | 비중 |
|---|---|---:|
| 전설적인 | +60% | 1.2% |
| 신성한 | +35% | 2.4% |
| 악마의 | +25% | 3.6% |
| 훌륭한 | +18% | 5.4% |
| 강력한 | +12% | 7.2% |
| 고통스러운 | +8% | 9.0% |
| 열성적인 | +6% | 9.0% |
| 날카로운 | +4% | 9.0% |
| 불안정한 | 이로운 값·해로운 값 모두 상승 (고위험/고보상) | 3.6% |
| 금이 간 | −12% | 8.4% |
| 하찮은 | −18% | 4.8% |
| 부서진 | −25% | 3.0% |

유물별 정확한 강화 수치는 인터랙티브 [`prefix_dashboard.html`](prefix_dashboard.html)에서 확인하세요.

---

## 2. 동반 유물 접두사

수치를 바꾸는 대신 **다른 유물의 효과를 통째로 부여**합니다. 능력치 유무와 무관하게 **어떤 유물에나** 붙을 수 있습니다. 효과는 host 유물 툴팁에 표시되고 발동 시 host 아이콘이 반짝입니다.

| 접두사 | 효과 | 원본 유물 | 비중 |
|---|---|---|---:|
| 가시돋친 | 전투 시작 시 가시 3 | 청동 비늘 | 5.4% |
| 강건한 | 전투 시작 시 힘 +1 | 금강저 | 3.6% |
| 수은의 | 매 턴 모든 적에게 3 피해 | 수은 모래시계 | 3.6% |
| 닻내린 | 전투 시작 시 블록 10 | 닻 | 4.2% |
| 피끓는 | 첫 턴에 체력 2 회복 | 피가 담긴 병 | 4.8% |
| 규칙적인 | 3턴마다 에너지 +1 | 행복한 꽃 | 3.0% |
| 통찰의 | 전투 중 첫 피격 시 카드 3장 드로우 | 백년퍼즐 | 4.2% |
| 위협적인 | 전투 시작 시 모든 적에게 취약 1 | 구슬 주머니 | 4.8% |

> 동반 접두사는 계속 추가될 예정입니다.

## 참고

- **밸런스 모드 아님** — 유물을 더 강하게 만드는 파워 판타지/캐주얼용.
- **저장/로드 유지** — 시드로 재유도되므로 세이브 후에도 동일.
- **모드 유물 지원** — 수치 접두사는 수치가 있는 모드 유물에 자동 적용.
- **비활성화** — ModConfig "접두사 미적용 확률"을 100%로.
