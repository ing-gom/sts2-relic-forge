# 遗物锻造 — 词缀指南

**语言:** [English](PREFIXES.en.md) · [한국어](PREFIXES.ko.md) · 简体中文

已锻造遗物可能获得的所有词缀。词缀由**种子锁定**（同种子 → 同结果）；初始遗物会被排除。默认仅约 **40%** 的遗物获得词缀（可用 ModConfig 滑块调节）。

> **占比**列是每个词缀**在词缀池中**的权重（当遗物获得词缀时抽取）。实际每件遗物的概率 ≈ 40% × 占比。

---

## 1. 数值词缀

提升或降低遗物的数值效果；提示框用颜色显示变化（绿色增加 / 红色减少）。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 传奇的 | +60% | 0.7% |
| 神级的 | +35% | 1.3% |
| 恶魔的 | +25% | 2.0% |
| 高级的 | +18% | 3.0% |
| 强力的 | +12% | 3.9% |
| 致伤的 | +8% | 4.9% |
| 狂热的 | +6% | 4.9% |
| 锐利的 | +4% | 4.9% |
| 不稳定的 | 增益与减益一并提升（高风险高回报） | 2.0% |
| 破损的 | −12% | 4.6% |
| 粗劣的 | −18% | 2.6% |
| 碎裂的 | −25% | 1.6% |

每件遗物的精确数值见交互式 [`prefix_dashboard.html`](prefix_dashboard.html)。

## 2. 同伴遗物词缀

这些词缀不强化数值，而是赋予另一件遗物的效果 —— 但是**弱化版**（数值降低，或触发更晚/更少），因此原版遗物仍值得获取。可出现在**任何**遗物上。

| 词缀 | 效果 | 源遗物 | 占比 |
|---|---|---|---:|
| 尖刺的 | 战斗开始时获得2荆棘 | 铜质鳞片 | 3.0% |
| 强壮的 | 第3回合获得1力量 | 金刚杵 | 2.0% |
| 水银的 | 每回合对所有敌人造成2点伤害 | 水银沙漏 | 2.0% |
| 沉稳的 | 战斗开始时获得6格挡 | 锚 | 2.3% |
| 血涌的 | 第1回合回复1点生命 | 小血瓶 | 2.6% |
| 规律的 | 每4回合获得1点能量 | 开心小花 | 1.6% |
| 洞察的 | 战斗中首次受击时抓2张牌 | 百年积木 | 2.3% |
| 威慑的 | 第2回合对所有敌人施加1易伤 | 弹珠袋 | 2.6% |
| 凶猛的 | 第1回合获得5鼓舞 | 赤牛 | 1.6% |
| 锋刃的 | 一回合内每4张技能牌对所有敌人造成3点伤害 | 开信刀 | 2.0% |
| 连击的 | 一回合内每4张攻击牌获得1力量 | 手里剑 | 1.6% |
| 淬火的 | 回合结束时若无格挡则获得4格挡 | 奥利哈钢 | 2.3% |
| 疾风的 | 一回合内每4张攻击牌获得2格挡 | 精致折扇 | 2.3% |
| 迅捷的 | 一回合内每4张攻击牌获得1敏捷 | 苦无 | 2.0% |
| 柔韧的 | 第2回合获得1敏捷 | 意外光滑的石头 | 2.3% |
| 加速的 | 每12张攻击牌获得1点能量 | 双截棍 | 1.6% |

> 更多同伴词缀即将推出。

## 3. 惩罚词缀

少数词缀是纯粹的负面效果（泰拉瑞亚式的坏词缀）—— 给遗物附加一个小诅咒，没有增益。出现概率低。

| 词缀 | 负面 | 占比 |
|---|---|---:|
| 被诅咒的 | 战斗开始时给予自己1虚弱 | 2.6% |
| 笨重的 | 第1回合自身敏捷-1 | 2.6% |
| 善变的 | 每回合25%概率给予自己1个随机减益 | 2.0% |
| 超载的 | 一回合内打出6张牌后给予自己1易伤 | 2.0% |
| 污秽的 | 每回合将1张眩晕加入抽牌堆 | 1.6% |
| 溃烂的 | 战斗开始时将2张伤口加入弃牌堆 | 1.6% |
| 阴燃的 | 战斗开始时将1张灼烧加入抽牌堆 | 1.6% |
| 虚空的 | 战斗开始时将1张虚无加入抽牌堆 | 1.6% |

## 4. 赌博词缀

也会作用于战场的混合效果 —— 可能有利（削弱敌人）或有害（削弱自己），每次都是掷硬币。敌方/玩家效果只作用于随机一个目标。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 侵蚀的 | 每回合使敌方一个能力值向零靠近1点（增益与减益皆可） | 2.0% |
| 暴露的 | 战斗开始时对一个敌人和自己施加1层易伤 | 2.0% |
| 衰弱的 | 战斗开始时对一个敌人和自己施加1层虚弱 | 2.0% |
| 混沌的 | 每回合50%概率：对一个敌人或一名玩家施加1层易伤或虚弱 | 2.0% |

---

## 5. 反应型词缀

在战斗中实时响应你的能力值/能量事件 —— 在你获得能力值或额外能量、或敌人试图削弱你的瞬间触发。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 共鸣的 | 获得力量或敏捷时，额外获得1点（每回合最多3次） | 1.6% |
| 顽固的 | 当敌人降低你的力量或敏捷时，反而获得等量 | 2.0% |
| 放电的 | 每当你获得额外能量时，对所有敌人造成4点伤害 | 2.0% |

---

## 说明

- **并非平衡模组** — 力量幻想/休闲向附加内容。
- **存读档保留** — 由种子重新推导，读档后不变。
- **支持模组遗物** — 数值词缀自动应用于有数值的模组遗物。
- **禁用** — 将 ModConfig “无词缀概率”滑块设为 100%。
