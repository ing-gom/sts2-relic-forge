# 遗物锻造 — 词缀指南

**语言:** [English](PREFIXES.en.md) · [한국어](PREFIXES.ko.md) · 简体中文

已锻造遗物可能获得的所有词缀。词缀由**种子锁定**（同种子 → 同结果）；初始遗物会被排除。默认仅约 **40%** 的遗物获得词缀（可用 ModConfig 滑块调节）。

> **占比**列是每个词缀**在词缀池中**的权重（当遗物获得词缀时抽取）。实际每件遗物的概率 ≈ 40% × 占比。

---

## 1. 数值词缀

提升或降低遗物的数值效果；提示框用颜色显示变化（绿色增加 / 红色减少）。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 传奇的 | +60% | 0.6% |
| 神级的 | +30% | 1.2% |
| 恶魔的 | +25% | 1.8% |
| 高级的 | +18% | 2.7% |
| 强力的 | +12% | 3.6% |
| 致伤的 | +8% | 4.5% |
| 狂热的 | +6% | 4.5% |
| 锐利的 | +4% | 4.5% |
| 不稳定的 | 增益与减益一并提升（高风险高回报） | 1.8% |

每件遗物的精确数值见交互式 [`prefix_dashboard.html`](prefix_dashboard.html)。

> **减益掷出 → 诅咒。** 过去约 **8%** 的掷出会得到*变弱*的词缀（破损 −12% / 粗劣 −18% / 碎裂 −25%）。如今这些掷出改为附加一个 **[诅咒](CURSES.zh.md)**——遗物保留原有数值并附带诅咒。所有负面效果都归入诅咒槽。

## 2. 同伴遗物词缀

这些词缀不强化数值，而是赋予另一件遗物的效果 —— 但是**弱化版**（数值降低，或触发更晚/更少），因此原版遗物仍值得获取。可出现在**任何**遗物上。

| 词缀 | 效果 | 源遗物 | 占比 |
|---|---|---|---:|
| 尖刺的 | 战斗开始时获得1荆棘 | 铜质鳞片 | 2.7% |
| 强壮的 | 第3回合获得1力量 | 金刚杵 | 1.8% |
| 水银的 | 每回合对所有敌人造成1点伤害 | 水银沙漏 | 1.8% |
| 沉稳的 | 战斗开始时获得3格挡 | 锚 | 2.1% |
| 血涌的 | 第1回合回复1点生命 | 小血瓶 | 2.4% |
| 规律的 | 每4回合获得1点能量 | 开心小花 | 1.5% |
| 洞察的 | 战斗中首次受击时抓1张牌 | 百年积木 | 2.1% |
| 威慑的 | 第2回合对所有敌人施加1易伤 | 弹珠袋 | 2.4% |
| 凶猛的 | 第1回合获得2鼓舞 | 赤牛 | 1.5% |
| 锋刃的 | 一回合内每4张技能牌对所有敌人造成1点伤害 | 开信刀 | 1.8% |
| 连击的 | 一回合内每4张攻击牌获得1力量 | 手里剑 | 1.5% |
| 淬火的 | 回合结束时若无格挡则获得1格挡 | 奥利哈钢 | 2.1% |
| 疾风的 | 一回合内每4张攻击牌获得1格挡 | 精致折扇 | 2.1% |
| 迅捷的 | 一回合内每4张攻击牌获得1敏捷 | 苦无 | 1.8% |
| 柔韧的 | 第2回合获得1敏捷 | 意外光滑的石头 | 2.1% |
| 加速的 | 每12张攻击牌获得1点能量 | 双截棍 | 1.5% |

> 更多同伴词缀即将推出。

## 3. 惩罚词缀 = 诅咒

少数词缀是纯粹的负面效果（泰拉瑞亚式的坏词缀）—— 给遗物附加一个小诅咒，没有增益。出现概率低。**它们被视为诅咒**（合并后的统一概念）：重铸出诅咒会**结束重铸**（营火与商店皆是——无法再用廉价金币重铸覆盖掉），唯一的清除方式是在商店**净化**，净化后遗物会还原为无词缀。详见[诅咒指南](CURSES.zh.md)。

| 词缀 | 负面 | 占比 |
|---|---|---:|
| 被诅咒的 | 战斗开始时给予自己1虚弱 | 2.4% |
| 笨重的 | 第1回合自身敏捷-1 | 2.4% |
| 善变的 | 每回合25%概率给予自己1个随机减益 | 1.8% |
| 超载的 | 一回合内打出6张牌后给予自己1易伤 | 1.8% |
| 污秽的 | 每回合将1张眩晕加入抽牌堆 | 1.5% |
| 溃烂的 | 战斗开始时将2张伤口加入弃牌堆 | 1.5% |
| 阴燃的 | 战斗开始时将1张灼烧加入抽牌堆 | 1.5% |
| 虚空的 | 战斗开始时将1张虚无加入抽牌堆 | 1.5% |

## 4. 赌博词缀

也会作用于战场的混合效果 —— 可能有利（削弱敌人）或有害（削弱自己），每次都是掷硬币。敌方/玩家效果只作用于随机一个目标。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 侵蚀的 | 每回合使敌方一个能力值向零靠近1点（增益与减益皆可） | 1.8% |
| 暴露的 | 战斗开始时对一个敌人和自己施加1层易伤 | 1.8% |
| 衰弱的 | 战斗开始时对一个敌人和自己施加1层虚弱 | 1.8% |
| 混沌的 | 每回合50%概率：对一个敌人或一名玩家施加1层易伤或虚弱 | 1.8% |

---

## 5. 反应型词缀

在战斗中实时响应你的能力值/能量事件 —— 在你获得能力值或额外能量、或敌人试图削弱你的瞬间触发。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 共鸣的 | 获得力量或敏捷时，额外获得1点（每回合最多3次） | 1.5% |
| 顽固的 | 当敌人降低你的力量或敏捷时，反而获得等量 | 1.8% |
| 放电的 | 每当你获得额外能量时，对所有敌人造成4点伤害 | 1.8% |

---

## 6. 局势词缀

响应你的局势状态（金币、卡组、诅咒牌），而非战斗事件。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 噬咒的 | 抽到诅咒牌时，获得1点力量或敏捷（每回合1次） | 1.5% |
| 镀金的 | 战斗开始时，每300金币获得1点力量 | 1.2% |
| 征税的 *(诅咒)* | 战斗开始时，每有1张牌失去1金币 | 1.8% |

---

## 7. 关键词词缀

通过游戏自身系统赋予卡牌关键词 —— 关键词直接显示在卡牌上。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 留存的 | 回合结束时，随机保留1张手牌 | 1.8% |
| 灼焚的 *(诅咒)* | 打出卡牌时，25%概率使其获得消耗（下次打出时生效） | 1.8% |
| 回响的 | 每回合，手牌中随机1张牌获得1层重演。打出该牌时，给予自己1层易伤和1层脆弱 | 0.9% |

## 8. 角色专属词缀

仅在使用该角色时出现，围绕角色的招牌机制（中毒/充能球/召唤/星星）。每个角色3个增益+2个诅咒。

| 角色 | 词缀 | 效果 | 占比 |
|---|---|---|---:|
| Silent | 剧毒的 | 对敌人施加中毒时，50%概率额外施加1层 | 2.2% |
| Silent | 飞刀的 | 打出小刀时，25%概率将1张小刀加入手牌 | 1.9% |
| Silent | 循环的 | 弃牌时，抓1张牌 | 1.4% |
| Silent | 自毒的 *(诅咒)* | 战斗开始时，给予自己3层中毒 | 1.9% |
| Silent | 钝毒的 *(诅咒)* | 你施加的中毒无法使敌人的中毒层数超过6 | 1.9% |
| Defect | 集束的 | 战斗中首次获得集中时，额外获得1点集中 | 1.9% |
| Defect | 增幅的 | 引导充能球时，25%概率额外引导1个随机充能球 | 1.6% |
| Defect | 过载的 | 充能球槽位填满时，获得1点集中（未填满则失去） | 1.6% |
| Defect | 短路的 *(诅咒)* | 战斗开始时，集中-2 | 1.9% |
| Defect | 两极的 *(诅咒)* | 回合结束时充能球槽全空或全满，则下回合失去1点能量 | 1.9% |
| Necrobinder | 死灵的 | 召唤时，获得3点格挡 | 2.2% |
| Necrobinder | 厄运的 | 对敌人施加厄运时，额外施加1层 | 1.9% |
| Necrobinder | 缚骨的 | 每回合召唤1（成长你的奥斯提） | 1.6% |
| Necrobinder | 献祭的 *(诅咒)* | 召唤物受到伤害时，给予自己1层虚弱/脆弱/易伤（每回合1次）；召唤物死亡时为2层 | 1.9% |
| Necrobinder | 被蚀的 *(诅咒)* | 每回合给予自己1层厄运 | 1.9% |
| Regent | 重税的 *(诅咒)* | 消耗星星的卡牌，星星费用+1 | 1.9% |
| Regent | 星耀的 | 每回合获得1颗星 | 1.9% |
| Regent | 重铸的 | 锻造卡牌时，获得1颗星 | 1.6% |
| Regent | 威严的 | 消耗星辰时，50%概率返还1颗星 | 1.6% |
| Regent | 破产的 *(诅咒)* | 每消耗4颗星，手牌中随机1张牌获得虚无 | 1.9% |

> 角色专属词缀的占比按该角色的词缀池（公共池 + 该角色5个）计算。

## 说明

- **并非平衡模组** — 力量幻想/休闲向附加内容。
- **存读档保留** — 由种子重新推导，读档后不变。
- **支持模组遗物** — 数值词缀自动应用于有数值的模组遗物。
- **禁用** — 将 ModConfig “无词缀概率”滑块设为 100%。
