# StS2 Relic Forge (유물 접두사)

상점과 보상으로 얻는 유물에 **테라리아식 접두사**를 붙여 수치를 강화하는 **슬레이 더 스파이어 2** 모드입니다 — `닻`이 **전설적인 닻**이 되어 더 많은 방어도를 줍니다. 가벼운 파워 판타지 모드예요.

## 무엇을 하나

- **접두사가 유물 수치를 강화** — 접두사가 유물의 수치 효과를 올리거나 내리고, 툴팁에 증감이 색으로 표시됩니다(초록 증가 / 빨강 감소).
- **희귀도 등급 · 시드 고정** — 등급은 런 시드로 결정되어 같은 시드 = 같은 결과. 저장/로드 후에도 유지됩니다.
- **등급 색 이름** — 강화된 유물의 이름 전체가 접두사 등급 색으로 물듭니다(전설적인 / 신성한 / 악마의 … 부서진).
- **자동 제외** — 시작 유물과 수치가 없는 유물은 자동으로 제외됩니다.
- **설정 가능** — ModConfig 슬라이더로 접두사가 안 붙을 확률을 조절합니다(기본 60% = 바닐라, 약 40%만 강화).

## 접두사

단일 테라리아식 풀 — 어떤 유물이든 어떤 접두사든 붙을 수 있습니다:

| 등급 | 접두사 | 효과 |
|---|---|---|
| 강화 | 전설적인, 신성한, 악마의, 훌륭한, 강력한, 고통스러운, 열성적인, 날카로운 | +60% … +4% |
| 불안정한 | 불안정한 | 이로운 값·해로운 값 모두 상승 (고위험·고보상) |
| 약화 | 금이 간, 하찮은, 부서진 | −12% … −25% (순화됨, 접두사 롤의 약 26%) |

등급은 시드 고정이라 같은 런 시드는 항상 같은 강화 결과를 냅니다. **접두사는 계속 추가될 예정입니다.**

## 작동 방식

모든 유물은 툴팁에 표시되는 값과 **같은 `DynamicVars`**에서 효과 크기를 읽습니다. 그래서 base 값 하나만 스케일하면 효과와 표시 숫자가 동시에 반영됩니다 — 유물별 코드 없이, 수치를 가진 모드 유물도 자동 지원됩니다. 하드코딩 보상 유물(잃어버린 궤짝, 니오우의 호부)은 별도 처리됩니다.

## 참고

- **밸런스 모드 아님** — 유물을 더 강하게 만드는 파워 판타지 / 캐주얼용입니다.
- **프로토타입** — 피드백 환영합니다.
- **비활성화** — ModConfig "접두사 미적용 확률" 슬라이더를 100%로 두면 순수 바닐라 유물이 됩니다.
- 지원 언어: English, 한국어, 简体中文.

## 설치

1. 최신 릴리스 zip을 받거나 Steam 창작마당에서 구독합니다.
2. `Sts2RelicForge/` 폴더를 `<Slay the Spire 2 설치 경로>/mods/` 에 넣어 아래처럼 되게 합니다:
   ```
   <Slay the Spire 2>/mods/Sts2RelicForge/Sts2RelicForge.dll
   <Slay the Spire 2>/mods/Sts2RelicForge/Sts2RelicForge.json
   ```
3. 게임을 실행합니다.

## 소스에서 빌드

요구사항: .NET SDK, Godot.NET.Sdk 4.5.1(자동 해석), 로컬 Slay the Spire 2 설치본, 그리고 같은 위치의 자매 프로젝트 `Sts2.ModKit`.

```sh
dotnet build Sts2RelicForge.csproj -c Debug
```

빌드하면 `Sts2RelicForge.dll` 과 `Sts2RelicForge.json` 이 `<sts2>/mods/Sts2RelicForge/` 로 복사됩니다.

`prefix_dashboard.html` 은 유물 × 접두사 결과를 전부 보여주는 독립 참조 페이지입니다.
