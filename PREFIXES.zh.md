# 遗物锻造 — 词缀指南

**语言:** [English](PREFIXES.en.md) · [한국어](PREFIXES.ko.md) · 简体中文

已锻造遗物可能获得的所有词缀。词缀由**种子锁定**（同种子 → 同结果）；初始遗物会被排除。默认仅约 **40%** 的遗物获得词缀（可用 ModConfig 滑块调节）。

> **占比**列是每个词缀**在词缀池中**的权重（当遗物获得词缀时抽取）。实际每件遗物的概率 ≈ 40% × 占比。

---

## 1. 数值词缀

提升或降低遗物的数值效果；提示框用颜色显示变化（绿色增加 / 红色减少）。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 传奇的 | +60% | 0.9% |
| 神级的 | +35% | 1.9% |
| 恶魔的 | +25% | 2.8% |
| 高级的 | +18% | 4.2% |
| 强力的 | +12% | 5.6% |
| 致伤的 | +8% | 7.0% |
| 狂热的 | +6% | 7.0% |
| 锐利的 | +4% | 7.0% |
| 不稳定的 | 增益与减益一并提升（高风险高回报） | 2.8% |
| 破损的 | −12% | 6.5% |
| 粗劣的 | −18% | 3.7% |
| 碎裂的 | −25% | 2.3% |

每件遗物的精确数值见交互式 [`prefix_dashboard.html`](prefix_dashboard.html)。

## 2. 同伴遗物词缀

这些词缀不强化数值，而是赋予另一件遗物的效果 —— 但是**弱化版**（数值降低，或触发更晚/更少），因此原版遗物仍值得获取。可出现在**任何**遗物上。

| 词缀 | 效果 | 源遗物 | 占比 |
|---|---|---|---:|
| 尖刺的 | 战斗开始时获得2荆棘 | 铜质鳞片 | 4.2% |
| 强壮的 | 第3回合获得1力量 | 金刚杵 | 2.8% |
| 水银的 | 每回合对所有敌人造成2点伤害 | 水银沙漏 | 2.8% |
| 沉稳的 | 战斗开始时获得6格挡 | 锚 | 3.3% |
| 血涌的 | 第1回合回复1点生命 | 小血瓶 | 3.7% |
| 规律的 | 每4回合获得1点能量 | 开心小花 | 2.3% |
| 洞察的 | 战斗中首次受击时抓2张牌 | 百年积木 | 3.3% |
| 威慑的 | 第2回合对所有敌人施加1易伤 | 弹珠袋 | 3.7% |
| 凶猛的 | 第1回合获得5鼓舞 | 赤牛 | 2.3% |
| 锋刃的 | 一回合内每4张技能牌对所有敌人造成3点伤害 | 开信刀 | 2.8% |
| 连击的 | 一回合内每4张攻击牌获得1力量 | 手里剑 | 2.3% |
| 淬火的 | 回合结束时若无格挡则获得4格挡 | 奥利哈钢 | 3.3% |
| 疾风的 | 一回合内每4张攻击牌获得2格挡 | 精致折扇 | 3.3% |
| 迅捷的 | 一回合内每4张攻击牌获得1敏捷 | 苦无 | 2.8% |
| 柔韧的 | 第2回合获得1敏捷 | 意外光滑的石头 | 3.3% |
| 加速的 | 每12张攻击牌获得1点能量 | 双截棍 | 2.3% |

> 更多同伴词缀即将推出。

---

## 说明

- **并非平衡模组** — 力量幻想/休闲向附加内容。
- **存读档保留** — 由种子重新推导，读档后不变。
- **支持模组遗物** — 数值词缀自动应用于有数值的模组遗物。
- **禁用** — 将 ModConfig “无词缀概率”滑块设为 100%。
