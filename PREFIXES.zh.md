# 遗物锻造 — 词缀指南

**语言:** [English](PREFIXES.en.md) · [한국어](PREFIXES.ko.md) · 简体中文

已锻造遗物可能获得的所有词缀。词缀由**种子锁定**（同种子 → 同结果）；初始遗物会被排除。默认仅约 **40%** 的遗物获得词缀（可用 ModConfig 滑块调节）。

> **占比**列是每个词缀**在词缀池中**的权重（当遗物获得词缀时抽取）。实际每件遗物的概率 ≈ 40% × 占比。

---

## 1. 数值词缀

提升或降低遗物的数值效果；提示框用颜色显示变化（绿色增加 / 红色减少）。

| 词缀 | 效果 | 占比 |
|---|---|---:|
| 传奇的 | +60% | 1.2% |
| 神级的 | +35% | 2.4% |
| 恶魔的 | +25% | 3.6% |
| 高级的 | +18% | 5.4% |
| 强力的 | +12% | 7.2% |
| 致伤的 | +8% | 9.0% |
| 狂热的 | +6% | 9.0% |
| 锐利的 | +4% | 9.0% |
| 不稳定的 | 增益与减益一并提升（高风险高回报） | 3.6% |
| 破损的 | −12% | 8.4% |
| 粗劣的 | −18% | 4.8% |
| 碎裂的 | −25% | 3.0% |

每件遗物的精确数值见交互式 [`prefix_dashboard.html`](prefix_dashboard.html)。

---

## 2. 同伴遗物词缀

这些词缀不强化数值，而是将**另一件遗物的效果整体赋予**你的遗物。可出现在**任何**遗物上。效果显示在宿主遗物的提示框中，触发时宿主图标会闪光。

| 词缀 | 效果 | 源遗物 | 占比 |
|---|---|---|---:|
| 尖刺的 | 战斗开始时获得3荆棘 | 铜质鳞片 | 5.4% |
| 强壮的 | 战斗开始时获得1力量 | 金刚杵 | 3.6% |
| 水银的 | 每回合对所有敌人造成3点伤害 | 水银沙漏 | 3.6% |
| 沉稳的 | 战斗开始时获得10格挡 | 锚 | 4.2% |
| 血涌的 | 第1回合回复2点生命 | 小血瓶 | 4.8% |
| 规律的 | 每3回合获得1点能量 | 开心小花 | 3.0% |
| 洞察的 | 战斗中首次受击时抓3张牌 | 百年积木 | 4.2% |
| 威慑的 | 战斗开始时对所有敌人施加1易伤 | 弹珠袋 | 4.8% |

> 更多同伴词缀即将推出。

---

## 说明

- **并非平衡模组** — 力量幻想/休闲向附加内容。
- **存读档保留** — 由种子重新推导，读档后不变。
- **支持模组遗物** — 数值词缀自动应用于有数值的模组遗物。
- **禁用** — 将 ModConfig “无词缀概率”滑块设为 100%。
