# Relic Forge — 저주 가이드

**언어:** [English](CURSES.en.md) · 한국어 · [简体中文](CURSES.zh.md)

강화된 유물은 일정 확률로 **저주** 하나를 추가로 얻습니다. 저주는 두 종류이며, 한 유물에는 **둘 중 하나만** 붙습니다:

- **적 저주** — 지니는 동안 **적이 강해집니다**(힘·판금·체력 등).
- **자기 저주** — **막지 못하고 피격당할 때** 당신이 벌점을 받습니다.

유물 이름 뒤에 〈저주〉로 표시되고, 툴팁에 정확한 효과가 나옵니다.

> **페널티 접두사도 저주입니다.** [페널티 접두사](PREFIXES.ko.md#3-페널티-접두사--저주)(저주받은·무거운·오염된…)는 통합된 부정 효과로, 아래의 모든 규칙에서 저주로 취급됩니다 — **재련을 종료**시키고 **정화**로 제거됩니다. 다만 페널티는 접두사 자체가 저주라서, 정화하면 접두사를 유지하는 대신 접두사 없음으로 되돌아갑니다.

> **약화 굴림도 저주입니다.** 유물은 더 이상 *약해지는* 접두사(옛 금이 간·하찮은·부서진)를 얻지 않습니다. 그 부정 효과는 이제 **저주**로 표현됩니다 — 유물은 원래 성능을 유지한 채 저주를 얻으며, 수치가 깎이지 않습니다. 모든 부정 효과는 저주 슬롯에 모입니다.

> **저주가 대장간을 조여오지만, 항상 멈추진 않습니다.** 한 모닥불·상점에서 재련하는 동안 저주가 뜰 때마다 "저주의 기운"이 짙어집니다. 1~4번째 저주는 각각 **10% → 25% → 50% → 80%** 확률로 그 곳의 재련을 끝내고, **5번째 저주는 반드시 종료**시킵니다. 이 횟수는 **장소별**이라 다음 모닥불·상점에서 리셋됩니다. 끝나기 전까진 계속 다시 굴릴 수 있고(상점은 골드 지불), 불이 식으면 상점 **정화**(기본 100골드)로만 저주를 제거합니다.

> **기본 켜짐** — ModConfig의 **"Enemy forge"** 로 조절합니다. 순수 파워 판타지로 즐기려면 **끄면** 됩니다.

---

## 동작 방식

- **강할수록 저주** — 유물이 접두사를 얻을 때 저주가 함께 붙을 수 있습니다. 확률은 **접두사가 강할수록 높아집니다** — 약한 강화는 거의 저주가 없고, **전설급(+60%)** 은 거의 항상 저주가 붙습니다(기준 ~33%, ModConfig 슬라이더로 조절).
- **둘 중 하나만** — 적 저주와 자기 저주는 **동시에 붙지 않습니다**. ModConfig **"Self-curse share"** 로 둘의 비율을 조절합니다(기본: 저주 중 자기저주 22% — 더 매운 쪽이라 소수로 유지).
- **중복 없음** — 페널티 접두사에는 적 저주·자기 저주가 *추가로* 붙지 않습니다. 그 자체가 이미 저주이기 때문입니다.
- **결정적** — 어떤 저주가 얼마나 붙는지는 런 시드로 정해집니다(같은 시드 = 같은 결과, 멀티플레이 동기 안전). 재련하면 모든 것을 다시 굴리지만, 저주가 뜰 때마다 재련이 **종료**될 확률이 오르므로(아래 참조), 무한정 다시 굴려 저주를 피할 수는 없습니다.

## 적 저주

강한 유물의 저주일수록 적이 더 강해집니다. 힘·방어 계열은 **엘리트·보스 전투**에서 발동하며, 저주를 여러 개 지니면 각 저주가 **서로 다른 적에게 하나씩 분산**됩니다(적이 하나면 한 적에게 모두).

**힘·방어 계열** (엘리트·보스)

| 저주 | 적이 받는 효과 | 발동 |
|---|---|---|
| 재앙 *(of Wrath)* | 힘 | 전투 시작 |
| 악의 *(of Malice)* | 판금 (매 턴 블록) | 전투 시작 |
| 원한 *(of Spite)* | 가시 (반격) | 전투 시작 |
| 폭군 *(of the Tyrant)* | 힘 + 판금 + 가시 | 전투 시작 |
| 피 *(of Bloodlust)* | 재생 (매 턴 회복) | 전투 시작 |
| 인공물 *(of Warding)* | 인공물 (당신 디버프 무효) | 전투 시작 |
| 버퍼 *(of Shielding)* | 버퍼 (다음 N타 무효) | 전투 시작 |
| 광란 *(of Frenzy)* | 힘 증가 | **3번째 턴마다** |

**체력 계열** (범위 내 모든 적)

| 저주 | 효과 | 대상 |
|---|---|---|
| 활력 *(of Vigor)* | 최대 체력 +20% | 일반 전투의 모든 적 |
| 비대 *(of Girth)* | 최대 체력 +12% | 모든 전투의 모든 적 |
| 거인 *(of the Titan)* | 최대 체력 +30% | 엘리트 |
| 영겁 *(of Eternity)* | 최대 체력 +25% | 보스 |

## 자기 저주

**막지 못한 피격마다** 발동하며, 다중타를 다 못 막으면 **타격 수에 비례**해 누적됩니다(블록으로 전부 막으면 발동하지 않음 — 잘 막을수록 유리).

| 저주 | 효과 (무방비 피격당) |
|---|---|
| 무력 *(Enfeebling)* | 자신에게 약화 1 |
| 균열 *(Cracking)* | 자신에게 손상 1 |
| 취약 *(Vulnerating)* | 자신에게 취약 1 |
| 혼미 *(Bewildering)* | 뽑을 더미에 현기증 1장 |
| 비참 *(Wretched)* | 약화·손상·취약 중 무작위 1 |

**저주는 계속 추가될 예정입니다.**

## 참고

- **끌 수 있음** — 기본 켜짐. ModConfig **"Enemy forge"** 를 끄면 모든 저주가 표시·발동하지 않습니다(유물 강화 자체는 그대로).
- **조절** — ModConfig **"Curse chance"**(저주 확률 기준값), **"Self-curse share"**(자기저주 비율), **"Enemy balance strength"**(적 강화 배율).
- **저주 시 재련 확률 종료** — 저주가 더 이상 재련을 즉시 멈추지 않습니다. 모닥불·상점에서 저주가 뜰 때마다 대장간의 불이 "식을" 확률이 오릅니다: 1~4번째는 **10% / 25% / 50% / 80%**, **5번째 저주는 반드시 종료**. 횟수는 장소별이며 떠나면 리셋됩니다. 저주는 상점 **정화**(기본 100골드)로만 제거합니다.
- **정화** — 적 저주·자기 저주는 접두사를 유지한 채 제거하고, 페널티 접두사(그 자체가 저주)는 통째로 정화해 유물을 접두사 없음으로 되돌립니다.
