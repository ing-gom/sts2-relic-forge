# Relic Forge — 저주 가이드

**언어:** [English](CURSES.en.md) · 한국어 · [简体中文](CURSES.zh.md)

강화된 유물은 일정 확률로 **저주**를 추가로 얻습니다 — 그 유물을 지니는 동안 **엘리트·보스가 강해집니다**. 유물 이름 뒤에 〈저주〉로 표시되고, 툴팁에 적이 정확히 무엇을 얻는지 나옵니다.

> **기본 켜짐** — ModConfig의 **"Enemy forge"** 로 조절합니다. 순수 파워 판타지로 즐기려면 **끄면** 됩니다.

---

## 동작 방식

- **저주 확률** — 유물이 접두사를 얻을 때 약 **33%** 확률로 저주가 함께 붙습니다(ModConfig 슬라이더로 조절). 페널티(저주) 접두사에는 붙지 않습니다.
- **적 강화** — 엘리트/보스 전투에서 **적 한 명**에게 저주 효과가 적용되며, 저주를 여러 개 지니면 그 효과들이 **한 적에게 중첩**됩니다.
- **강해진 만큼** — 강한 유물의 저주일수록 적도 더 강해집니다. 수치는 **시드 고정 범위 랜덤**(같은 시드 = 같은 결과, 멀티플레이 동기 안전).
- **결정적** — 어느 적이 강해지고 수치가 얼마인지는 런 시드로 정해집니다.

## 저주 목록

| 저주 | 적이 받는 효과 | 발동 |
|---|---|---|
| 재앙 *(of Wrath)* | 힘 | 전투 시작 |
| 악의 *(of Malice)* | 판금 (매 턴 블록) | 전투 시작 |
| 원한 *(of Spite)* | 가시 (반격) | 전투 시작 |
| 폭군 *(of the Tyrant)* | 힘 + 판금 + 가시 | 전투 시작 |
| 피 *(of Bloodlust)* | 재생 (매 턴 회복) | 전투 시작 |
| 인공물 *(of Warding)* | 인공물 (당신 디버프 무효) | 전투 시작 |
| 버퍼 *(of Shielding)* | 버퍼 (다음 N타 무효) | 전투 시작 |
| 광란 *(of Frenzy)* | 힘 증가 | **3번째 턴마다** |

**저주는 계속 추가될 예정입니다.**

## 참고

- **끌 수 있음** — 기본은 켜짐. ModConfig **"Enemy forge"** 를 끄면 저주가 표시·발동하지 않습니다(유물 강화 자체는 그대로).
- **강도 조절** — ModConfig **"Enemy balance strength"** 로 적 강화 배율을, **"Enemy-rider chance"** 로 저주 확률을 조절합니다.
- **재련** — 유물을 재련하면 저주도 다시 굴립니다.
